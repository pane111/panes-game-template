# Pane11's Game Template

A Godot project containing some useful features to build a game with.
Features include:
- Player character with animations
- NPCs with different movement patterns
- Interactables
- Autoloads with functions like moving specific entities through commands or spawning objects
- A resource file type for characters, making it easy to set them up for dialogue (which is part of a plugin, so you can also use only that)
- Scene transitioning
- Saving/Loading data
- Audio settings menu

I made this for myself so I wouldn't have to keep redoing the same things over and over, but you're welcome to use this project for anything you see fit as long as you credit me.

Intended to be used with [Nathan Hoad's Dialogue Manager](https://github.com/nathanhoad/godot_dialogue_manager), although it's not required
